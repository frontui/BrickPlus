• BrickPlus.Uitl.currecy在金额体系有太多的金额数字，大多时候后端返回的往往是数字，而页面上产品需要显示千位符的金额文字，此时我们需要对金额数字进行格式化。
参数说明
属性 说明 类型 默认值
num 将格式化的数字 Number null
decimal 格式化小数点位数 Number 2

实例
          
// -----------
// 金额格式化
var fnCurrency = BrickPlus.Util.currency;
$(document).on('keyup', '#demo-currency-input', function() {
  var number = $(this).val();
  var $target = $('#demo-currency-target');

  $target.html(fnCurrency(number))

})
