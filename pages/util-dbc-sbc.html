• 用户在文本框或文本域中输入内容时，不经意输入全角字符，导致后端插库后全角占两个字符，数据无法检索等情况，为了避免输入全角字符问题，需要通过函数转换字符
  • 全角字符需要过滤转换的包裹数字和标点符号
  • 全角字符集 。．，；：？！—～〔〕【】｛｝《》‘’“”１２３４５６７８９０／＼＠＃＄％
  • 对应半角字符集 ..,;:?!-~()[]{}<>\'\'""1234567890/\@#$%
  • 使用BrickPlus.Util.dbcToSbc(str)进行转换返回转换后的字符串
  • 使用data-toggle="dbc2sbc"进行全局绑定插件。

实例一 - javascript调用
            
//---------------
// 全角半角转换
$(document).on('keyup', '#demo-util-dbc-input', function() {
  // 去除空格
  var dbcOriginText = $.trim($('#origin-text'))
  var thisVal = $(this).val()
  // 转换
  var filterVal = BrickPlus.Util.dbcToSbc(thisVal)
  // 显示原输入字符
  dbcOriginText.html(thisVal)
  // 重新赋值
  $(this).val(filterVal)
})

实例二 - data-toggle绑定
            
<input type="text" data-toggle="dbc2sbc" class="form-control" placeholder="请输入您的账号...">
