{# page: 日期天数间隔 #}
  • 计算多少天后的日期,dateDiff(date, num, prefix)
参数说明
属性 说明 类型 默认值
date 开始时间，必须符合日期格式xxxx-xx-xx String
num 计算天数 Number
prefix 格式拼接字符 String -

实例


0 
          
$(function() {
  var $start = $('#demo-date-diff-start'),
      $end = $('#demo-date-diff-end'),
      $text = $('#demo-date-diff'),
      dateDiff = BrickPlus.Util.dateDiff;

  var calcDate = function(a, b) {
    var days = dateDiff(a, b);
    $text.html(days)
  };

  $end.on('change', function() {
    var num = Math.floor($(this).val());
    var date = $start.val();
    calcDate(date, num);
  })
})
