• 金额简繁体转换在支付平台上使用比较广泛，将输入的数字金额转换成中文繁体大写。
参数说明
属性 说明 类型 默认参数
target 显示繁体的 dom 元素 String(jQuery selector) null
prefix 格式化的繁体文字前缀字符串 String 人民币
dec 小数点位数，转成中文的分，厘等 Number 2
unit 金额单位后缀 String 整

实例一 - javascript调用
            
//----------------
//  金额简繁体转换
var cnMoney = BrickPlus.Util.cnMoney,
    cnMoneyInput = $('#demo-util-cnmoney-input');
var IntanceCnMoney = new cnMoney(cnMoneyInput, {target: '#cnmoney-trandition'});
